# EuroMD To CARESM Data Conversion

This repository contains a script for converting EuroMD data to the CARE-SM semantic model. It includes:

- Functions to transform EuroMD data into CARE-SM compliant format.
- Functions to add synthetic data to demonstrate full coverage of CDE Properties in the semantic model.

## Usage

Run main script:

```
python3 main.py nameOfyourEuroMDDataset.csv
```
